CHANGE IS NEWS

We believe CHANGE ITSELF IS NEWS. While traditional media reports on events, we reverse the perspective: prediction changes ARE the news.

🎯 Our Approach

We track Polymarket prediction events in real-time, monitoring probability shifts across key thresholds (70% / 80% / 90%). When significant changes occur, we generate news.

📊 Event Categories

  • 📅 Daily Watch - Threshold Crossings: Events crossing 70%/80%/90% thresholds today, with history tracking
  • 🏆 Long-Term High-Probability Events: Stable events (70%-99% probability, excluding 100%) with TECH tag, volume ≥ 10K, end date ≥ today

📋 Selection Rules

  • ✅ TECH Tag: Events must have TECH tag (extensible to other tags)
  • ✅ Probability Range: 70%-99% (excluding 100% events)
  • ✅ Trading Volume: ≥ 10,000 USDC
  • ✅ End Date: ≥ Today (future events only)
  • ✅ Event Merging: Multiple crossings of same event are merged into one card

🔮 Data Source

All data comes from the Polymarket API (gamma-api.polymarket.com). Daily Watch updates hourly, Long-Term updates daily at 17:00 (Beijing Time).

🔮 Future Vision

Starting with technology events, we plan to expand to economy, culture, and more. Our unique "change as news" perspective offers a fresh take on prediction market analysis.
